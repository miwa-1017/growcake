###  🍰 **GrowCake**-日々の努力でケーキが完成していくアプリ

## サイト概要
ユーザーは、毎日の筋トレメニューを記録することでケーキを育てていきます。  
1日ごとにケーキが少しずつ完成していき、継続した日数に応じて達成感を得られる仕組みになっています。  

## サイトテーマ
筋トレの継続を”ケーキの成長”として可視化するモチベーション管理アプリ。  
ストイックなトレーニングを、かわいく楽しく続けられる世界を目指します。  
​
### テーマを選んだ理由
私はこれまで、モチベーションに波があり努力を継続することが苦手でした。そこで、頑張る過程をもっと楽しく感じられるように「成長を可視化する」仕組みを考えました。  
ケーキが少しずつ完成していく演出を取り入れることで、達成感を日々感じられるアプリを目指しています。  

​
## ターゲットユーザ-
・習慣化や学習の継続が苦手な人  
・日々の努力を可視化して達成感を感じたい人  
・見た目の変化でモチベーションを保ちたい人  
​
## 主な利用シーン
・毎日の筋トレ達成をチェックするとき  
・継続の成果を視覚的に実感したいとき  
・SNSで「今日のケーキ」を共有したいとき
